# Personal-Finance-Coach

 This repository showcases cutting-edge AI applications built using System Prompts, User Prompts, Tuning Parameters, Structured Outputs, Function Calling, and RAG (Retrieval-Augmented Generation).


### 🚀 Featured Projects

AI Research Assistant – Summarizes papers and extracts insights.

Smart Travel Planner – Generates custom itineraries with real-time data.

Legal Document Analyzer – Interprets contracts and flags risks.

Medical Diagnosis Helper – Suggests possible conditions (not a doctor!).

Personal Finance Coach – Optimizes budgets and spending habits.

### 🛠️ Core Technologies Used
Gemini (System & User Prompts, Tuning)

Function Calling (APIs: Google Maps, PubMed, Plaid)

RAG 

Structured Outputs (JSON, Markdown Tables)
